# IT_ONE
Навигатор оптимизации: Система анализа производительности процессов
Проанализировал кредитный процесс банка, выявил узкие места и создал ТЗ для системы мониторинга производительности. Решение поможет оптимизировать критически важные процессы.
В рамках задачи необходимо:
Провести анализ AS-IS процесса: изучить предоставленную BPMN-модель кредитного процесса и датасет с временными метриками, выявить критические узкие места и проблемные участки.
Разработать функциональные требования: определить ключевые показатели производительности (KPI), сформулировать бизнес-правила расчета метрик, описать требования к периодичности расчетов (ежедневно и по запросу).
Проектирование дашборда и отчетов: спроектировать визуализацию ключевых метрик, макеты пользовательских интерфейсов для разных ролей и формы отчетов для руководства с описанием сценариев использования.
Написать техническое задание на систему мониторинга производительности, где необходимо проработать следующие элементы:
откуда и как система будет получать данные (настройка подключения к базе данных),
с какой периодичностью происходят выборки логов,
как рассчитывается время каждого шага,
каким образом эти данные собираются в сводные показатели (почасовая, ежедневная и недельная агрегация),
где и в каком виде хранятся уже посчитанные метрики,
как формируются отчеты по расписанию и по запросу,
как и когда отправляются email-уведомления при сбоях или превышении порогов,
как система сохраняет и сравнивает «до» и «после» любых изменений в процессе,
и  возможность ретроспективного анализа и A/B-тестирования.
Проработать модель данных и ETL: описать модель хранения метрик производительности, ETL-процессы обработки логов и интеграцию с российскими BI-системами через стандартные коннекторы.
